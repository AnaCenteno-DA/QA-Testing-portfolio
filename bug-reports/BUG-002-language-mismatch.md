
# 🐞 Bug Report – Language Issue

## ID
BUG-002

## Title
Error messages not translated into selected language

## Description
When the system is set to French or Spanish, error messages still appear in English.

## Steps
1. Change language to French
2. Trigger login error
3. Observe message

## Expected
Message should appear in selected language

## Actual
Message appears in English

Medium

## Status
