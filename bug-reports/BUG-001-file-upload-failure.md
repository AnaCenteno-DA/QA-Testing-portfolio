

# 🐞 - Bug Report – File Upload Failure

## ID
BUG-001

## Title
File upload fails for files larger than 50MB

## Environment
- Browser: Chrome
- OS: Windows 11
- Module: File Upload

## Description
When a user uploads a file larger than 50MB, the upload fails without proper handling.

## Steps to Reproduce
1. Go to Upload section
2. Select file > 50MB
3. Click Upload

## Expected Result
File should upload or show a clear validation message

## Actual Result
Upload fails with generic error

## Severity
High

## Status
Open
